KEY ARTIFACTS

Mobile app config: Root configuration for Now Mobile. It groups navigation, theme, app behavior, and child configuration records.
Navigation bar: Container for bottom tabs. In the demo, the Now Mobile Nav contains Home, Support, Notifications, and Settings.
Navigation tab: A tab shown at the bottom of the mobile app. It usually points to a launcher screen or special tab function.
Launcher screen: Landing screen with header, search, and UI sections. Example: Home and Support.
Record section: A section on a launcher screen showing cards or results. Example: Recommended for you, Popular topics, Browse all topics.
Search config: Defines search behavior, source list, empty states, result limit, and suggestion limit.
List screen: A screen that displays a list of records; usually backed by an item stream.
Item stream: Defines the record stream/table and possibly interleaving/order behavior.
Data item: The data source for a stream or screen.
List item config: Defines how each row is rendered and when the configuration applies.
Card / card template: Visual template for rows/cards, edited in Mobile Card Builder.
Record screen: A detail page for one record such as a RITM, request, topic, or approval.
Screen segment: A subsection inside a screen, such as Details, Updates, Activity, or field areas.
Function: An action or navigation behavior. Examples: open catalog item, open topic page, jump to RITM, approve, reject, approve with comment.
